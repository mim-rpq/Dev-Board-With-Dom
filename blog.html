Question-1: What are the different ways to select an element in the DOM?

In the DOM, we can select elements in various ways using JavaScript.

  • Selecting by ID

    • document.getElementById("Id")

  • Selecting by Class Name

    • document.getElementsByClassName("Class");

  • Selecting by Tag Name

    • document.getElementsByTagName("p");

  • Selecting by Query Selector

    • document.querySelector(".Class")

  • Selecting by Query Selector All

    • document.querySelectorAll("div");

  • Selecting by Name Attribute

    • document.getElementsByName("username");

    Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • InnerHTML

    • innerHTML is a property that allows you to get or set the HTML content inside an element. It parses HTML, meaning it treats HTML tags as actual elements instead of plain text.

  • InnerText

    • innerText retrieves or sets the visible text inside an element. It ignores HTML tags and excludes hidden text

  • TextContent

    • textContent retrieves or sets all text inside an element, even if it's hidden with CSS. It does not parse HTML, treating it as plain text.

    What is event delegation in the DOM?

    Event Delegation is a technique in JavaScript that allows us to efficiently manage events on multiple child elements using a single event listener on a parent element. Instead of attaching an event listener to each individual child element, we place the listener on a common parent and use event bubbling to handle the event when it reaches the target element.
    It is a powerful technique that improves efficiency, performance, and scalability in JavaScript event handling. By leveraging event bubbling, we can manage events more effectively with minimal code.

    What is event bubbling in the DOM?

    Event Bubbling is a mechanism in the DOM where an event triggered on a child element first executes on that element and then propagates upward to its parent and ancestors.
    It makes event handling efficient, but sometimes we may need to stop it to prevent unintended behavior. Understanding bubbling helps in better event management and performance optimization in JavaScript.

    How do you create, add, and remove elements using JavaScript?

    Manipulating the DOM dynamically using JavaScript allows for creating interactive and responsive web applications. By leveraging methods like createElement(), appendChild(), and remove(), we can efficiently create, add, and remove elements as needed. Using event delegation and efficient DOM manipulation improves performance, especially when dealing with a large number of elements. Choosing the right method based on the situation ensures smooth and optimized web development. Mastering these techniques will help us build dynamic and user-friendly applications with ease.
